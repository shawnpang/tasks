# Submit AllScale to No-KYC / Crypto Directories

## What
Find and submit AllScale (non-custodial, permissionless neobank — no upfront KYC) to directories similar to kycnot.me.

## Why
Increase visibility in the no-KYC and privacy-focused crypto community. Drive organic discovery from users actively looking for permissionless financial tools.
